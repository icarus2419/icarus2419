My portfolio leans into the same idea I bring to software: make it dependable, make it clear, and give it enough personality that people remember it.

## Run The 3D Portfolio Locally

```bash
npm install
npm run dev
```

Open the Vite URL shown in the terminal, usually:

```text
http://localhost:5173
```

For a production build:

```bash
npm run build
npm run preview
```
